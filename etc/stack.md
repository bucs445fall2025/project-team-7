# Technology Stack

This document outlines the core technologies and Docker images used in the Borrow & Lend project. Each component links to its official Docker Hub page for documentation and setup instructions.

## **Backend**

- **Node.js (Alpine)** → [node:20-alpine](https://hub.docker.com/_/node)

## **Frontend**

- **Node.js (Alpine)** → [node:20-alpine](https://hub.docker.com/_/node)

## **Database**

- **MySQL** → [mysql:8.0](https://hub.docker.com/_/mysql)

## **Additional Dependencies**

- Express.js - Web framework for backend API
- React - Frontend UI library
- Prisma - ORM for database management
- TypeScript - Programming language for backend
- bcryptjs - Password hashing
- jsonwebtoken - JWT authentication
- Zod - Schema validation
